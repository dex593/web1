Bước 1

Nhập API key và endpoint

Sau khi xác thực, app sẽ hiện thông tin tài khoản và danh sách truyện có quyền đăng chapter.

Hướng dẫn nhanh

  1. Nhập API endpoint và API key, bấm Kết nối.
  2. Chọn truyện trong danh sách truyện bạn có quyền upload.
  3. Bấm Chọn thư mục mẹ chứa các folder chapter (vd: 184, 185).
  4. Kiểm tra bảng, chỉnh tiêu đề chapter nếu cần.
  5. Nếu chapter đã tồn tại, chọn Bỏ qua hoặc Up đè.
  6. Bấm Upload hàng loạt để chạy tiến trình.

Trạng thái khi đang upload hiển thị x/n + vòng xoay. Xong chapter sẽ hiện Hoàn thành.
